Introducción

Las páginas HTML del Configurador MFE y de Pricing emplean para su diseño y maquetación el framework responsive Foundation versión 5. En su web se puede encontrar información más detallada y completa de todos y cada uno de los componentes que se han empleado para maquetar las distintas páginas del Configurador MFE y de Pricing. Ambas webs comparten los estilos CSS y la mayoría de los componentes.

En cuanto a los componentes dinámicos basados en javascript, se hace uso de jQuery v2.1.1 además de la librería javascript del propio Foundation. Esta última librería permite, como es lógico, cargar selectivamente los componentes que se van a utilizar, permitiendo economizar recursos. También se han añadido algunos scripts externos para cubrir funcionalidades solicitadas que no se incluyen en el framework (calendario, multiselect, autocompletar, data slider y tablas ordenables)

En la navegación de cada página es posible acceder secuencialmente, mediante la tecla tabulador, a todos los elementos interactivos (enlaces y elementos de formulario principalmente) que aparecerán resaltados al recibir el foco. Asímismo, al cargarse una página con un formulario, el foco se sitúa en el primer elemento del mismo, si se trata de un input type="text" con un atributo value no vacío, el texto aparece seleccionado.

Según los requerimientos del proyecto las páginas se adaptan al tamaño del dispositivo empleado, ya sea el monitor de un ordenador de sobremesa, una tableta o un smartphone (estos últimos pueden presentar algún desajuste por no ser requisito). En monitores con una resolución superior a 1280px la web se muestra en el navegador centrada y con una anchura máxima de 1320px.

En la navegación izquierda se listan los enlaces a las distintas partes de esta guía.
